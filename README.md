# TouchDesigner Workspace

A collection of TouchDesigner sketches and reusable components.

## Structure

- `_template/` - Base template for new projects
- `components/` - Reusable `.tox` components
- `sketch_YYYYMMDD/` - Daily sketch projects

## Components

- `ComponentTemplate.tox` - Template for creating new components
- `KinectRefinedDepthmap.tox` - Kinect depth map processing
